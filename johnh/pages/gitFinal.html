GitHub Final

Github is a site that allows you upload your code, control versions, and overall share your code.

These are steps to upload your code to GitHub

1. git install
2. $git init
- show .git file
3. $git config --global
< user.name >
< user.email >
$git config --get
4. $git status
5. $touch .gitignore
6. $git add -A
7. $git commit -m "short message"
8. $git remote add origin < GitHub URL >
9. $git push master
