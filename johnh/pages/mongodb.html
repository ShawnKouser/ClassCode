mongodb find, filter, and projection by john houser

db.collectionName.find(); 
Output: lists items in collection

This is how you find all items in a mongodb 

db.products.find({"price": {$gt: 5.00, $lt: 9.00}});
Output: shows items greater than 5 and less than 9

This is how you would filter a search (in this case it would find items more expensive than 5 but less expensive than 9) 

db.products.find({"price": {$gt: 5.00, $lt: 9.00}},{_id:0, description:0});
Output: shows items greater than 5 and less than 9 but does not include the fields id and description

The second part of the expression represents the projection. This would not include the id elements and the description.
