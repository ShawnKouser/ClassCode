					+----+------+---+
				A*N	|    |      |   |
				F.A	|    |      |   |
					+----+------+---+

A reg ex crossword is designed to help you understand how Regular Expressions work. Solving this type of crossword is very simple. All you have
to do is match box with the correct letter that matches the horizontal and vertical regular expression. Try to see if you can solve it yourself (it's easy!) 

					 *+	DA   .*
					+----+------+---+
				A*N	| A  |  D   | N |
				F.A	| F  |  A   | A |
					+----+------+---+

Here is the solution. If you got the solution wrong, make sure you understand that each box needs to meet the requirements of both regular expressions that align with that
box. Sometimes, there are even 2 horizontal and 2 vertical requirements! Again, it's the same idea but it is just more requirements!
